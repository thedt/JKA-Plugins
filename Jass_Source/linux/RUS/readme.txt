Описание:
JASS - Jedi Academy Server Security - (в переводе с анг. "Защита сервера Академии Джедаев", сокр. "Джаз"). Коротко говоря, джаз - проект, направленный на защиту серверов на так называемом базовом моде - "basejka". Так же увеличивается защита и других модов. Основан на исходном коде QMM. Он представляет из себя несколько динамических библиотек. Исходный код джаза открыт, но исходники защитного модуля не будут опубликованы. Список исправленных уязвимостей:
1) msgbof.
2) votebof.
3) infobof.
4) forcecrash.
5) teamcrash.
6) votebug/cbufexec.
7) dirtrav.
8) npccrash.
9) limitbug.
10) gcrash.
11) RCON disabler.
12) RCON bruteforce.
13) Fake Players DDoS.
14) RandomPacket DDoS.
15) UnBan.
16) Удалены все возможности физической порчи дистрибутива игры — к примеру, при помощи writeconfig.
17) Удалена возможность «убить сервер» — выключить его так, что скрипт рестарта не поможет.
18) ChatFlood (donedl)
19) Ну и мелочи.
Для того, чтобы узнать список доступных команд джаза, введите в консоли сервера команду "jass".
Формат: jass <команда> [параметры]
Поддерживаемые комманды:
    status - информация о JASS.
    list - информация о загруженных плагиных.
    load <file> - загрузка нового плагина.
    info <id/str> - информация о плагине.
    pause <id/str> - остановка плагина.
    unpause <id/str> - запуск плагина.
    unload <id/str> - выгрузка плагина.
    force_unload <id/str> - принудительная выгрузка плагина.
    shutdown - отключение ДЖАЗа, за исключением функций, отвечающих за поддержание работы.
Следующие команды будут доступны, только если загружен плагин AntiDDoS:
    ban <IP/range> <minutes> - банит IP через JASS. Диапазон IP указывается через дефис.
    unban <IP/range> - удаляет бан IP через JASS. Ведите "all" для полной очистки списка.
    banlist [list_num] [count] - список забаненных.

Установка:
    1. Откройте папку с модом, в которую вы хотите поместить JASS (обычно это base)
    2. Найдите в папке с модом jampgamei386.so. Переименуйте в jass_jampgamei386.so.
    3. Извлеките из архива JASS файлы jass.so, jassconfig.cfg и папку plugins в
       папку с модом ("base", "japlus", "MBII", "clanmod", "lugormod", "makermod" и т.д.).
    4. Переименуйте jass.so в jampgamei386.so.
    5. Извлеките из архива JASS jass.ini и pdb.so в папку Gamedata.
    6. Настройте jass.ini. Для этого откройте его текстовым редактором (notepad, notepad++, gedit), и следуйте подсказкам в комментариях. Обратите внимание, что для установки JASS на моды (т.е. не на base), отредактировать jass.ini необходимо. В противном случае можно оставить как есть.
	
Настройка:
В файле jass.ini настройки для каждой папки с установленным джазом прописываются отдельно, поэтому изначально в файле дан блок настроек только для папки base.
Блок настроек выглядит так:
#Имя папки
"название_папки"{
  Настройки
}
Для установки джаза в другие папки нужно скопировать данный блок настроек и сменить имя с "base" на что-либо другое, к примеру, "japlus", "lugormod", "MBII". Обратите внимание, что при невыполнении данного шага защита работать не будет, т.к. плагины грузиться не будут. Все строки, в начале которых написан символ #, игнорируется джазом и используется для комментариев. Следуя комментариям, вы можете настроить джаз. Каждая настройка выглядит, как:
"имя_настройки" "значение";
или
"имя_настройки" 'Числовое_значение';
Значение можно менять. Любые настройки можно удалять, тогда будут использованы стандартные.

Установка плагинов:
Изначально в джазе два плагина - AntiDDoS и Protection. Оба плагина составляют основное предназначение джаза - защиту сервера. Эти плагины присутствиую в настройках изначально, но, т.к. jass.ini содержит стандартные настройки лишь для base, изначально эти плагины так же грузятся только для basejka. Т.е. для установки защиты на моды необходимо Аналогично можно подключить и другие плагины. Для этого в секции "plugins" нужно вписать адрес до плагина:
		"plugins/plugin.so";
Чтобы убедиться, что плагин загружен, введите jass list в консоли сервера.
